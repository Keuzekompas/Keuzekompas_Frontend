# Project: Keuzekompas_Frontend

## Project Overview

This is a Next.js project bootstrapped with `create-next-app`. It's a web application built with React and TypeScript. The project uses Tailwind CSS for styling and ESLint for linting. The configured font is Geist.

## Building and Running

To get started, you'll need to install the dependencies:

```bash
npm install
```

Then, you can run the development server:

```bash
npm run dev
```

This will start the development server at `http://localhost:3000`.

### Other Commands

*   **`npm run build`**: Builds the application for production.
*   **`npm run start`**: Starts a production server.
*   **`npm run lint`**: Lints the project using ESLint.

## Development Conventions

*   **TypeScript**: The project is written in TypeScript. Adhere to the configurations in `tsconfig.json`.
*   **Styling**: Tailwind CSS is used for styling.
*   **Linting**: ESLint is set up. Run `npm run lint` to check for issues.
*   **Path Aliases**: The project uses path aliases. Imports from the root of the project can be prefixed with `@/`. For example, `import MyComponent from '@/components/MyComponent';`.
